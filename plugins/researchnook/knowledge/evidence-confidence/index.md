# Evidence and confidence labeling

Use this playbook whenever a Researchnook phase turns source material
into analysis or report language.

## Evidence labels

| Label | Meaning | May support final judgment? |
|-------|---------|-----------------------------|
| `fact` | Directly checkable statement, date, event, policy, or definition | yes |
| `data` | Numeric indicator, statistic, sample, or measurement | yes, with source and scope |
| `quoted_opinion` | Third-party expert, institution, or user opinion | yes, but not as fact |
| `assumption` | Required premise that has not been verified | only with caveat |
| `inference` | Intermediate conclusion drawn from evidence | yes, if chain is shown |
| `judgment` | Final recommendation or report conclusion | yes, only after evidence chain |

## Confidence scale

| Level | Use when |
|-------|----------|
| High | Multiple independent sources agree; scope and timing are clear; counterexamples checked |
| Medium | Evidence is plausible but incomplete, dated, narrow, or partly inferential |
| Low | Evidence is sparse, user-provided only, unverified, or mostly assumption-driven |

## Forecast caveats

Forecasting phases must include:

1. base / upside / downside scenarios;
2. trigger conditions that would move the report from one scenario to another;
3. sensitive variables;
4. explicit uncertainty and confidence;
5. a statement that the report is not investment, purchase, legal, or financial advice.

## Claim wording

- Use "suggests", "is consistent with", or "under these assumptions" for medium confidence.
- Use "could", "may", or "scenario" for forecast language.
- Do not use "will", "guarantees", or "proves" unless the evidence truly supports it.

## Minimum final report caveat

> This report is a structured research aid, not investment, purchase,
> legal, financial, or professional advice. Conclusions depend on the
> sources, assumptions, time horizon, and uncertainty listed above.
